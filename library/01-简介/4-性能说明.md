# 性能说明

## 性能指标

| 指标         | 单位   | 说明                                   |
| ---          | ---    | ---                                    |
| 链上总事务数 | 个     | 区块链上所有区块的事务总数             |
| 单个事务大小 | 字节   | 单个事务在区块链中所占用的存储空间大小 |
| 连接数       | 个     | 系统同时接受的连接数量                 |
| 系统启动时间 | 秒     | 系统从被执行到可接受请求的时间长度     |
| 事务处理速度 | 个/秒  | 每秒系统可以处理的事务数量             |
| 查询吞吐量   | 次/秒  | 每秒系统可以处理并完成返回的查询数量   |
| 内存占用     | 兆字节 | 系统占用内存的数量                     |
